DNA-LCEB Server

Usage Notes

  1. Capacity Calculation.
    • Choose a sequence from list or Enter a sequence in the given field.
    • Enter corresponding genetic code (if not known insert 1) for the sequence.
    • In case the sequence is chosen from list, genetic code is automatically retrieved.
    • Submit the form pressing Calculate. If the operation is successful, you will be redirected to a page with stating capacity in number of bits and characters, otherwise error will be displayed. 16 bits out of available capacity are reserved for supplementary data with watermark.

  2. Data embedding.
    • Choose a sequence from list or Enter a sequence in the given field.
    • Enter corresponding genetic code (if not known insert 1) for the sequence.
    • In case the sequence is chosen from list, genetic code is automatically retrieved.
    • Enter a test message
    • Submit the form pressing Embed. If the operation is successful, you will be redirected to a page with resulting watemarked sequence, or error will be displayed.

  3. Data Extraction.
    • Insert watermarked sequence in given field.
    • Enter corresponding genetic code (if not known insert 1) for the sequence.
    • Submit the form pressing Extract. If the operation is successful, you will be redirected to a page with extracted watermark message, otherwise error will be displayed.
